# Spam Email Classifier using NLP

## Overview
This project classifies emails as spam or non-spam using Natural Language Processing and machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLP (Text Vectorization)

## Dataset
Spam email dataset sourced from Kaggle.

## Approach
- Text preprocessing
- Feature extraction using TF-IDF
- Model training using Naive Bayes
- Model evaluation

## Model Used
- Multinomial Naive Bayes

## Conclusion
The model successfully classifies spam and non-spam emails, demonstrating the application of NLP techniques in text classification tasks.
